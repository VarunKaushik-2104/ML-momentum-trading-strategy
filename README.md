# Machine Learning Momentum Trading Strategy

This project implements a machine learning based momentum trading system developed for the SPARQ Quant Hackathon.

## Features

- Feature engineering from OHLCV financial data
- Ensemble Voting Classifier for stock return prediction
- Weekly portfolio allocation strategy
- Backtesting with transaction cost simulation
- Performance metrics including Sharpe ratio and drawdown

## Dataset

Historical OHLCV data for major Indian stocks (2015–2025).

## Outputs

- Weekly portfolio weights
- Predicted stock probabilities
- Portfolio returns
- Performance metrics
